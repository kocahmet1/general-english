2. If I **were** you, I would **accept** the offer. *(were + would + base verb)*
3. If she **studied** harder, she would pass. *(Past Simple in IF clause)*
4. If I had **known**, I would have told you. *(Past Participle: known)*
5. I would have helped if you **had asked**. *(Past Perfect, not "would have")*
6. If it **rains** tomorrow, I **will stay** home. *(Present Simple + will)*
7. If I **were** rich, I would buy a house. *(Past Simple, not "would be")*
8. She would **have passed** if she had studied. *("would have" + V3)*
